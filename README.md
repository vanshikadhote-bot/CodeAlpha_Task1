# CodeAlpha_Task1
A simple text-based Hangman game built in Python where players guess a hidden word one letter at a time. The game uses random word selection, loops, conditional statements, and lists while allowing up to 6 incorrect guesses.
# CodeAlpha_Task2
A Python application that tracks stock investments by calculating the total portfolio value based on user-entered stock quantities and predefined stock prices. It demonstrates the use of dictionaries, user input, and basic arithmetic operations.
# CodeAlpha_Task4
A rule-based chatbot developed in Python that responds to predefined user inputs such as greetings and common questions. The project demonstrates conditional statements, loops, functions, and user interaction.
